# OnTrackTab Privacy Policy

Last updated: May 4, 2026

OnTrackTab is a local-first Chrome new tab extension. It replaces Chrome's new tab page with a customizable dashboard for search, widgets, notes, weather, calendar, frequently visited sites, and wallpapers.

## Data Stored Locally

OnTrackTab stores the following data locally in your browser:

- Extension settings and widget preferences
- Grid layout and appearance preferences
- Local notes
- Liked wallpapers
- Cached Unsplash wallpaper data
- Optional Tomorrow.io API key
- Optional Unsplash Access Key
- Cached weather data

This data is used only to provide the extension's user-facing features.

## Browser Data Access

OnTrackTab requests Chrome permissions for:

- Bookmarks, to provide bookmark-first search.
- History and top sites, to show frequently visited sites on the new tab page.
- Storage, to save local settings and cached extension data.

Bookmark, history, and top-site data are used locally by the extension for the features shown in the user interface. OnTrackTab does not sell this data and does not transmit it to the developer.

## Optional External Services

OnTrackTab can optionally connect to external APIs if you add the relevant API key:

- Tomorrow.io, for live weather data.
- Unsplash, for live wallpaper photos.

Requests to these services are made only when the related feature is enabled. These requests are sent over HTTPS.

## Data Sharing

OnTrackTab does not sell user data. OnTrackTab does not transfer user data to the developer. Optional API requests are sent directly from the extension to the selected third-party service only when enabled by the user.

## Contact

For support or privacy questions, use the support contact listed on the Chrome Web Store listing or the project repository.
