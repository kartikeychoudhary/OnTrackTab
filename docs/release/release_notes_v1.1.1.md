# ZenBoard v1.1.1 Release Notes

## GitHub Release Notes

ZenBoard v1.1.1 is a branding and store-readiness update. The extension has been renamed from OnTrackTab to ZenBoard across the app, manifest, documentation, and release materials.

## What Changed

- Renamed the extension from OnTrackTab to ZenBoard in the Chrome extension manifest and in-app surfaces.
- Updated app copy, README references, export metadata, sample notes, and visible UI labels to use ZenBoard.
- Added the new ZenBoard logo as the extension icon set and in-app brand mark.
- Added the logo to the topbar greeting area and Settings About panel.
- Created Chrome Web Store promo assets for the small promo tile and marquee tile.
- Moved the full-resolution logo to a shared public asset path for reuse by the app and promo HTML.

## Chrome Web Store Description Note

ZenBoard v1.1.1 completes the rename from OnTrackTab to ZenBoard, refreshes the extension icon and in-app branding, and adds updated Chrome Web Store promotional artwork.
